Ako implementovať kód sledovania

Vložte tento skript za značku head na svojom webe.


  <script>
  (function(i,s,o,g,r,a,m){i['StorePredictorObjectV2']=r;i[r]=i[r]||function(){
  (i[r].q=i[r].q||[]).push(arguments)},i[r].l=1*new Date();a=s.createElement(o),

  })(window,document,'script','https://cdn.storepredictor.com/sp.v2.js','sp');

  sp('create', 'YOUR_SP_CODE', 'auto');
  </script>

Vložte tento skript pre koncocovu značku tagu body

Pre trackovanie stránok ktore nereprezentujú detail produktu ani nákupný proces (ako napr. homepage, kontakt form a pod.)

  <script>sp('send', 'pageview', '');</script>

Pre trackovanie stránky s detailom produktu vlože za predchadzajúci kód ešte tento kód.

  <script>sp('send', 'pageview', 'PRODUCT_CODE');</script>

Tento skript vložte do stránky, ktorá sa zobrazí zákazníkovi po úspešnom dokončení objednávky (tzv. thank you page).


  <script>
  sp('create', 'YOUR_SP_CODE', 'auto');
  sp('send', 'order', { orderId: 'ORDER_NUMBER', items:   [ { productCode: 'PRODUCT_CODE', unitPrice: 100, quantity: 1 } ], totalPrice: 100, currency: 'EUR' });
  </script>
