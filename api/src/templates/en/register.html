How to implement tracking code

Insert this script after the head tag on your site


          <script>
          (function(i,s,o,g,r,a,m){i['StorePredictorObjectV2']=r;i[r]=i[r]||function(){
          (i[r].q=i[r].q||[]).push(arguments)},i[r].l=1*new Date();a=s.createElement(o),
          m=s.getElementsByTagName(o)[0];a.async=1;a.src=g;m.parentNode.insertBefore(a,m)
          })(window,document,'script','https://cdn.storepredictor.com/sp.v2.js','sp');

          sp('create', 'YOUR_SP_CODE', 'auto');
          </script>

Insert this script before closing end tag of your body.

For tracking homepage or other sites without product detail.

          <script>sp('send', 'pageview', '');</script>

For tracking product detail page add to previous call one more call.

          <script>sp('send', 'pageview', 'PRODUCT_CODE');</script>

Insert this script to your order confirmation page.


          <script>
          sp('create', 'YOUR_SP_CODE', 'auto');
          sp('send', 'order', {
                orderId: 'ORDER_NUMBER',
                items:   [ { productCode: 'PRODUCT_CODE', unitPrice: 100, quantity: 1 } ],
                totalPrice: 100, currency: 'EUR' });
          </script>
